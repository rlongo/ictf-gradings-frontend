How to format the Students List File

  • You must have exactly 4 columns, no extras
  • The first row of the excel file must have the headers (spelled exactly the same):
    • First Name Last Name Belt Size Belt Level
  • Belt Size can be either a number ("2") or the word size and a number "size 00"
  • Belt level should be LITTLE DRAGON COLOUR STRIPE/BELT. If they aren't a little dragon, then: COLOUR STRIPE/BELT. Some examples of valid inputs: "Little Dragon Blue Stripe", "Red Stripe", "Blue Belt"
